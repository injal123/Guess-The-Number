# Guess The Number



## Description:
**Guess The Number** is an interactive web-based game built with HTML, CSS, and JavaScript,
that challenges users to guess a randomly generated number between 0 and 100.   
This project helps practice DOM manipulation, event handling, and basic game logic.


## Tech Stacks:
- **HTML5** – For page structure  
- **CSS3** – For styling and layout  
- **JavaScript** – For game logic, DOM updates, and interactivity
- **Git & GitHub** – Version control and repo hosting  
- **GitHub Pages** – For live deployment


## How to Play:
1. Enter a number between **0 and 100**.  
2. Click **Guess**.  
3. The game provides feedback:  
   - "Your number is greater than actual no."  
   - "Your number is smaller than actual no."  
   - "Congrats, you got it right!"  
4. You have **10 tries** to guess the correct number.  
5. All previous guesses are displayed so you can keep track.


## Features:
- Input validation to prevent invalid entries  
- Dynamic feedback based on user input  
- Remaining attempts counter  
- Previous guesses displayed for reference  
- Visual reward upon successful guess
